# Krevat Zoom Link

This application was created to make it easier for my family to find the Zoom link for our weekly Zoom meetings. With this application, the host can update the Zoom link and everyone else can simply click a button to open the meeting, given they provide the correct passcode. 

## Stack

### Backend

* [Node.js](https://nodejs.org/en/)
* [Express.js](https://expressjs.com/)
* [TypeScript](https://www.typescriptlang.org/)
* [MongoDB](https://www.mongodb.com/)
* [mongoose.js](https://mongoosejs.com/)

### Frontend

* [Vue.js](https://vuejs.org/)
* [Buefy](https://buefy.org/)
* [SCSS](https://sass-lang.com/)

### Deployment

* [Heroku](https://www.heroku.com/)
* [MongoDB Atlas](https://www.mongodb.com/cloud/atlas/)
